# Image Intel - תדריך האקתון

## המשימה

בניית מערכת חילוץ מודיעין מתמונות. המערכת מקבלת תיקיית תמונות ומייצרת דו"ח מודיעיני מקיף: מיקומים על מפה, ציר זמן, זיהוי דפוסים וקשרים.

## רקע - מהו EXIF?

כל תמונה שמצולמת בטלפון או מצלמה מכילה מידע נסתר שנקרא EXIF (Exchangeable Image File Format). המידע הזה כולל:

- **מיקום GPS** - קו רוחב וקו אורך של מקום הצילום
- **תאריך ושעה** - מתי בדיוק צולמה התמונה
- **מכשיר** - יצרן ודגם המצלמה/טלפון
- **הגדרות צילום** - פוקוס, חשיפה, ISO ועוד

מידע זה הוא "מכרה זהב" מודיעיני - מתמונה אחת אפשר לדעת איפה מישהו היה, מתי, ועם איזה מכשיר.

## מה המערכת עושה?

1. שליפת metadata מכל התמונות
2. הצגת כל המיקומים על מפה אינטראקטיבית
3. יצירת ציר זמן של הצילומים
4. זיהוי קשרים ודפוסים (אותו מכשיר, אותו מיקום, קרבה בזמן)
5. הרכבת דו"ח HTML יפה עם כל המידע

## הארכיטקטורה

```
תמונות
  │
  ▼
extractor.py ──→ רשימת מילונים עם כל המידע
  │
  ├──→ map_view.py ──→ מפה (HTML)
  ├──→ timeline.py ──→ ציר זמן (HTML)
  ├──→ analyzer.py ──→ תובנות ודפוסים
  │
  ▼
report.py ──→ דו"ח HTML מלא
  │
  ▼
app.py (Flask) ──→ ממשק ווב
```

## מבנה הצוותים

### רביעייה 1 - Data (4 תלמידים)

| זוג | מודול | אחריות |
|-----|-------|---------|
| A | extractor.py | שליפת EXIF מתמונות (GPS, תאריך, מכשיר) |
| B | map_view.py | יצירת מפה אינטראקטיבית עם folium |

הזוגות ברביעייה עוזרים אחד לשני ועושים Code Review הדדי.

| B | analyzer.py | זיהוי דפוסים (אותו מכשיר, אותו מיקום, קרבה בזמן) |
