# Image Intel - מערכת חילוץ מודיעין מתמונות

## מה זה?

מערכת שמקבלת תמונות ומייצרת דו"ח מודיעיני: מיקומים על מפה, ציר זמן, זיהוי דפוסים וקשרים.

## התקנה

```bash
git clone <repo-url>
cd image_intel
pip install -r requirements.txt
```

## הרצה

```bash
python src/app.py
```

גשו ל-`http://localhost:5000` בדפדפן.

## מבנה הפרויקט

```
image_intel/
```
